## 7.

Write a function `flip` that takes object as a parameter and returns a new object with keys that are values from original object and values that are keys.

For example:

```js
var result = flip({
  a: 1,
  b: 2,
  c: 3
});

console.log(result); // { 1: 'a', 2: 'b', 3: 'c' }
```
